# Example app with Jest tests inside a NextJS TypeScript app

### This app included:
```node
(1) apisauce
(2) ramda
(3) react-i18next
(4) react-redux
(5) redux-persist

```

### The structure of project:
```
pages
src/actions
src/components
src/i18n
src/reducers
src/sagas
src/screens
src/selectors
src/services
src/store
src/utils
```

### Install it and run:

```bash
npm install
npm run dev
# or
yarn
yarn dev
```

## Run Jest tests

```bash
npm run test
# or
yarn test
```

## The idea behind the example

This example shows a configuration and several examples for a running Jest tests in a NextJS TypeScript app
